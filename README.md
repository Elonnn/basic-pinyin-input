# crude-pinyin
> 简陋的拼音输入法
- 命令行界面下输入和选择
- 通过读入pinyin.txt和word.txt两个资源文件加载音节表和词库
- 全拼输入（必要时需加'作为分隔符）
- 可选择记录和读入用户细胞词库
